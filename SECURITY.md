# Security Policy

## Supported Versions

NodeJS versions:

| Version | Supported          |
| ------- | ------------------ |
| 15      | :white_check_mark: |
| 14      | :white_check_mark: |
| 12      | :white_check_mark: |
| 10      | :white_check_mark: |
| < 10    | :x:                |

## Reporting a Vulnerability

If you want reporting a vulnerablity you can open an [issue](https://github.com/kamitoonapp/naoki/issues) or talk about this in [discord support server](https://discord.gg/X9kfP4kam6)
